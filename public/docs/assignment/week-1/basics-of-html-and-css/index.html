    Basics of HTML and CSS#

    Disclaimer#

    All the steps and explanations in this documentation are written specifically for users on the Windows operating system.
    While Git itself is cross-platform, certain installation steps, paths, and UI elements may differ on Linux or macOS.

    Choosing a Text Editor#

    Before working with Git or documentation files, a text editor is required. A text editor is essential for:

    • Writing documentation (Markdown / HTML)
    • Editing configuration files
    • Managing project files efficiently

    Some commonly used text editors on Windows include:

    • Visual Studio Code
    • Sublime Text
    • Notepad++
    • Atom

    Selecting a Text Editor#

    For my workflow, I decided to use Visual Studio Code (VS Code) because:

    • I am already familiar with it
    • It has a clean and intuitive interface
    • It offers excellent Git integration
    • It supports extensions for almost every language

    However, you are free to choose any text editor you are comfortable with.

    Downloading Visual Studio Code#

    To download Visual Studio Code:

    1. Open Google Chrome
    2. Go to the official Visual Studio Code website
    3. Download the latest Windows .exe installer

    Always make sure to download the installer from the official website.

    Installing Visual Studio Code#

    During installation, most default options can be kept unchanged.
    However, ensure that the following option is selected:

    • ✅ Add “Open with Code” action to Windows Explorer

    This allows Visual Studio Code to be opened directly from the File Explorer using the right-click menu.

    After installation, Visual Studio Code is ready for use.

    Installing Useful VS Code Extensions#

    After installing Visual Studio Code, the next step is to install a few helpful extensions that make web development and documentation easier.

    Live Server Extension#

    The Live Server extension allows you to:

    • Launch a local development server
    • Automatically refresh the browser when files are saved

    Steps to install:

    1. Open Visual Studio Code
    2. Go to the Extensions tab (Ctrl + Shift + X)
    3. Search for Live Server
    4. Click Install

    Save Typing / Auto Save#

    To avoid losing changes and to improve workflow, Auto Save can be enabled:

    1. Click on File
    2. Select Auto Save

    This ensures files are saved automatically while typing.

    Getting Started with HTML#

    With the editor and extensions ready, the next step is to try the basics of HTML (HyperText Markup Language).

    Creating a New Project in VS Code#

    1. Open Visual Studio Code
    2. Click on File → Open Folder
    3. Create a new folder and give it a name (for example: html-basics)
    4. Select the folder and click Open

    This folder will act as your project workspace.

    Creating a New HTML File#

    1. Inside the opened folder, create a new file
    2. Name the file index.html

    Basic HTML Structure#

    Below is a basic HTML template:

    <!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

    Brief Explanation of the Code#

    <!DOCTYPE html>#

    Declares the document type and ensures proper rendering in modern web browsers.

    <html>#

    The root element of an HTML page.

    <head>#

    Contains metadata about the document, such as the page title.

    <title>#

    Sets the title shown on the browser tab.

    <body>#

    Contains all the visible content of the webpage.

    Common HTML Elements#

    Headings and Paragraphs#

    • <h1> to <h6> are used for headings (largest to smallest)
    • <p> is used for paragraphs

    Example:

    <h1>Main Heading</h1>
<p>This is a paragraph of text.</p>

## Images in HTML

Images are added using the `<img>` tag:

```html
<img src="image.jpg" alt="Sample Image">
    • src specifies the image file path
    • alt provides alternative text if the image cannot be displayed

    Hyperlinks#

    Links are created using the <a> tag:

    <a href="https://fabacademy.org">Fab Academy</a>
    • href specifies the URL of the link destination

    Introduction to Basic CSS#

    CSS (Cascading Style Sheets) is used to style HTML elements.

    Example:

    <style>
  body {
    background-color: lightgray;
  }
  h1 {
    color: blue;
  }
</style>

    This CSS changes the background color of the page and the color of the heading.
