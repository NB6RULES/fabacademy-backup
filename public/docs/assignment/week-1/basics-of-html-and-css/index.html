    Basics of HTML and CSS#

    Disclaimer#

    All the steps in this guide are written for Windows users because I use Windows. Git works on other systems too, but some things may look different on Linux or Mac.

    Choosing a Text Editor#

    Before starting with Git or making websites, we need a text editor. A text editor helps us to:

    • Write code (HTML, CSS, Markdown)
    • Edit files
    • Organize our project

    Some popular text editors on Windows are:

    • Visual Studio Code
    • Atom

    Selecting a Text Editor#

    I chose Visual Studio Code because:

    • I already know how to use it
    • It looks clean and easy
    • It works well with Git
    • It supports many programming languages

    You can use any editor you like.

    Downloading Visual Studio Code#

    To download VS Code:

    1. Open Google Chrome
    2. Go to the official Visual Studio Code website
    3. Download the Windows .exe installer

    Always download from the official website to stay safe.

    Installing Visual Studio Code#

    During installation, most options can stay the same.

    But make sure this option is checked:

    • ✅ Add “Open with Code” action to Windows Explorer

    This helps open folders easily in VS Code by right-clicking.

    After this, VS Code is ready to use.

    Installing Helpful Extensions#

    Extensions make coding easier.

    Live Server Extension#

    Live Server helps to:

    • Open your website in a browser
    • Automatically refresh when you save files

    Steps:

    1. Open VS Code
    2. Go to the Extensions (Ctrl + Shift + X)
    3. Search for Live Server
    4. Click Install

    Save Typing / Auto Save#

    To avoid losing work:

    1. Click on File
    2. Select Auto Save

    This ensures files are saved automatically while typing.

    Getting Started with HTML#

    HTML is used to create websites.

    Creating a New Project in VS Code#

    1. Open VS Code
    2. Click on File → Open Folder
    3. Create a new folder and give it a name (for example: html-basics)
    4. Open it

    This folder will store your project files.

    Creating a New HTML File#

    1. Create a new file
    2. Name it index.html

    Basic HTML Structure#

    <!DOCTYPE html>
<html>
<head>
  <title>Page Title</title>
</head>
<body>

  <h1>This is a Heading</h1>
  <p>This is a paragraph.</p>

</body>
</html>

    What This Code Means#

    <!DOCTYPE html>#

    tells the browser this is an HTML file

    <html>#

    starts the webpage

    <head>#

    contains page info

    <title>#

    shows the name on the browser tab

    <body>#

    contains what we see on the page#

    Common HTML Elements#

    Headings and Paragraphs#

    • <h1> to <h6> are used for headings (largest to smallest)
    • <p> is used for paragraphs

    Example:

    <h1>Main Heading</h1>
<p>This is a paragraph of text.</p>

## Images in HTML

Images are added using the `<img>` tag:

```html
<img src="image.jpg" alt="Sample Image">
    • src= image location
    • alt= text if image doesn’t load

    Hyperlinks#

    Links are created using the <a> tag:

    <a href="https://fabacademy.org">Fab Academy</a>
    • href specifies the URL of the link destination

    Introduction to Basic CSS#

    CSS (Cascading Style Sheets) is used to make websites look nice.

    Example:

    <style>
  body {
    background-color: lightgray;
  }
  h1 {
    color: blue;
  }
</style>

    This changes the background color and heading color.
